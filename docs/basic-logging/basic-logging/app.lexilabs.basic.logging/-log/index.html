An easy-to-use method of logging.

Use Log.v, Log.d, Log.i, Log.w, Log.e, and Log.wtf methods to write to logcat.

Parameters

tag

typically the name of the parent class.

message

whatever string of information you'd like to pass.

class RandomClass() {

  val tag = "RandomClass"

  fun test() {
    Log.v(tag, "it works!")
  }
}
